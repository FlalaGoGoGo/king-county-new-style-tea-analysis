# Data

This public repository includes a cleaned project data package.

- `data/reference/`: dashboard-ready tables used by the published site
- `data/processed/`: intermediate outputs used in scoring and ranking
- `data/raw/`: selected source tables that are safe to share publicly
- `data/interim/`: small helper tables used by selected scripts

Large raw exports, local caches, API-error backups, and full raw review pools are intentionally excluded.
